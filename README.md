# Masicalan
Masicalan は，マクロ記述などを目的とした，F# + .NET で動くミニマルな構成のスクリプト言語です．

## 開発環境
Microsoft Windows 11 Insider Preview Experimental<br>
Visual Studio 2026<br>
.NET 10.0

## 使用しているライブラリ
パーサの実装には [FParsec](https://github.com/stephan-tolksdorf/fparsec) を利用しています．<br>
